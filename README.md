<h1 align="center">hfetch</h1>
<p align="center">A tiny and lightweight fetch written in POSIX sh</p><br>

Characteristics
-
* Tiny and lightweight
* Written in **POSIX sh**
* Simple

Compatibility
-
* 100% Compatible in **GNU/Linux** distros.
* 100% Compatible in any shell (sh, bash, dash)
* 100% Compatible with **glibc**
* 100% Compatible with **musl libc**
* 100% Compatible with POSIX.
* 100% Compatible in any architecture (x86, x86_64, arm, arm64)
